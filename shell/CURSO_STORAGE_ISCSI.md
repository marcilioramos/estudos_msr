### Teoria ISCSI
~~~
ISCSI: Internet Small Computer System Interface
Target: Servidor
Initiators: Clientes
Luns: Blocos de dados, como discos fisicos.
Rede SAN: Rede Storage Area Network
IQN: ISCSI Qualified Name
  - iqn."data"."dominio"."identificador_qualquer"
Segurança:
  - Texto puro
  - Sem autenticação
  - CHAP
~~~
